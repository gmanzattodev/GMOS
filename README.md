# 💻 GM OS - Portfólio

> Um portfólio interativo inspirado na experiência de um sistema operacional.

O **GM OS** ainda está em desenvolvimento. Novas funcionalidades, melhorias na interface e novos projetos serão adicionados continuamente.

---

## 📖 Sobre

O **GM OS** é um portfólio pessoal criado para apresentar meus projetos, habilidades e minha evolução como desenvolvedor de uma forma criativa e interativa.

A interface é inspirada em um sistema operacional, oferecendo uma experiência semelhante a um desktop, com janelas, aplicativos e animações.

---

## ✨ Funcionalidades

- 🖥️ Interface inspirada em um sistema operacional
- 📁 Área de trabalho (Desktop)
- 📌 Barra de tarefas
- 🚀 Menu Iniciar
- 🪟 Sistema de janelas
- 👨‍💻 Página Sobre Mim
- 📂 Portfólio de Projetos
- 📞 Página de Contato
